# وكيل الذكاء الاصطناعي لـ Termux (Termux AI Agent)

## الوصف
هذا المشروع عبارة عن وكيل ذكاء اصطناعي متكامل مصمم للعمل في بيئة Termux على أجهزة أندرويد. يهدف الوكيل إلى محاكاة تجربة وكلاء الذكاء الاصطناعي المتقدمين المتوفرين على أنظمة لينكس المكتبية، مما يوفر للمستخدمين القدرة على أتمتة المهام، تنفيذ أوامر Shell، إدارة الملفات، والوصول إلى معلومات الجهاز، بالإضافة إلى تصفح الويب، كل ذلك من خلال واجهة نصية تفاعلية.

## الميزات
*   **تكامل مع نماذج اللغة الكبيرة (LLMs):** يدعم نماذج مثل GPT-4o (عبر OpenAI API) ويمكن توسيعه لدعم نماذج أخرى.
*   **تنفيذ أوامر Shell:** القدرة على تنفيذ أي أمر في Termux والحصول على النتائج مباشرة.
*   **إدارة الملفات:** قراءة وكتابة وتعديل الملفات داخل نظام ملفات Termux.
*   **تصفح الويب:** البحث في الإنترنت وجلب المعلومات من صفحات الويب.
*   **تكامل Termux API:** الوصول إلى ميزات الجهاز مثل حالة البطارية ومعلومات Wi-Fi.
*   **واجهة مستخدم تفاعلية (TUI):** باستخدام مكتبة `rich` و `prompt_toolkit` لتجربة مستخدم محسنة.
*   **نظام أدوات قابل للتوسيع:** بنية معيارية تسمح بإضافة أدوات ووظائف جديدة بسهولة.

## المتطلبات
*   جهاز أندرويد مثبت عليه تطبيق Termux.
*   اتصال بالإنترنت.
*   مفتاح API صالح لـ OpenAI (أو أي نموذج لغة كبير آخر يتم استخدامه).

## التثبيت
1.  **افتح Termux** على جهاز الأندرويد الخاص بك.
2.  **استنسخ المستودع:**
    ```bash
    git clone https://github.com/abdabdahfa2002/termux-ai-agent.git
    cd termux-ai-agent
    
3.  **اجعل سكربت التثبيت قابلاً للتنفيذ وقم بتشغيله:**
    ```bash
    chmod +x setup.sh
    ./setup.sh
    ```
4.  **قم بتحرير ملف `.env`:**
    سيقوم سكربت التثبيت بإنشاء ملف `.env` في مجلد المشروع. افتحه باستخدام محرر نصوص (مثل `nano` أو `vim`) وأضف مفتاح API الخاص بك:
    ```bash
    nano .env
    ```
    استبدل `your_api_key_here` بمفتاح OpenAI API الخاص بك:
    ```
    OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
    MODEL_NAME=gpt-4o
    ```
    احفظ الملف واخرج.

## الاستخدام
بعد اكتمال التثبيت وتكوين مفتاح API، يمكنك تشغيل الوكيل:

```bash
python agent.py
```

أو بعد إعادة تشغيل Termux، يمكنك استخدام الاختصار:

```bash
termux-ai
```

يمكنك الآن التفاعل مع الوكيل عن طريق كتابة الأوامر والأسئلة. سيقوم الوكيل بتحليل طلباتك واستخدام الأدوات المتاحة له لتنفيذ المهام.
